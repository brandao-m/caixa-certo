# CaixaCerto

Sistema financeiro para autônomos, desenvolvido para ajudar profissionais a controlarem clientes, serviços, receitas, despesas e lucro de forma simples e organizada.

## Objetivo

O CaixaCerto foi idealizado como uma aplicação full stack para auxiliar profissionais autônomos no controle financeiro do dia a dia, centralizando informações importantes do negócio em um só lugar.

## Funcionalidades planejadas

- autenticação de usuários
- cadastro de clientes
- cadastro de serviços
- registro de atendimentos realizados
- controle de receitas
- controle de despesas
- dashboard com resumo financeiro
- filtros por período

## Stack do projeto

### Backend
- FastAPI
- SQLModel
- PostgreSQL
- Alembic
- JWT

### Frontend
- React
- Vite
- Tailwind CSS
